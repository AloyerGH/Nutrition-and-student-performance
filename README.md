# Опросник «Питание и успеваемость студентов»

Учебный проект по дисциплине «Учебная практика», выполненный в рамках летней практики 2026 г.

## Описание

Одностраничный веб-опросник на Python + Streamlit с сохранением ответов в облачную базу данных Firebase Firestore и визуализацией собранных данных.

**Тема:** Питание и успеваемость  
**Ключевые метрики:** режим питания, частота употребления фастфуда, влияние на концентрацию и успеваемость

## Структура проекта

```
survey_app/
├── app.py                  # Основное приложение Streamlit
├── serviceAccountKey.json  # Ключ Firebase (не публиковать!)
├── requirements.txt        # Зависимости Python
├── .gitignore              # Исключения для Git
└── README.md               # Документация
```

## Требования

- Python 3.8+
- Google-аккаунт с настроенным проектом Firebase (Firestore)

## Установка и запуск

**1. Клонировать репозиторий:**
```bash
git clone <ссылка на репозиторий>
cd survey_app
```

**2. Установить зависимости:**
```bash
pip install -r requirements.txt
```

**3. Добавить ключ Firebase:**  
Положить файл `serviceAccountKey.json` в корень папки проекта.  
Файл можно скачать в Firebase Console → Настройки проекта → Сервисные аккаунты → Создать новый закрытый ключ.

**4. Запустить приложение:**
```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

## Функциональность

### Форма опроса (4 раздела, 13 вопросов)

| Раздел | Вопросы |
|--------|---------|
| Общая информация | Возраст, курс, средняя успеваемость |
| Режим питания | Количество приёмов пищи, пропуск завтрака, соблюдение режима |
| Фастфуд | Частота, причины, предпочтительные напитки |
| Концентрация и учёба | Самооценка концентрации, связь питания и продуктивности, барьеры |

### Аналитический дашборд

Доступен по чекбоксу «Показать аналитику». Включает:

- Сводная таблица всех ответов
- Метрики: количество ответов, средняя концентрация, средний возраст
- Гистограмма самооценки концентрации
- Барчарт частоты употребления фастфуда
- Box-plot: концентрация vs уровень успеваемости
- Pie-chart: частота пропуска завтрака
- Барчарт: осознанность связи питания и продуктивности
- Кнопка экспорта данных в CSV

## База данных

Данные сохраняются в коллекцию `nutrition_survey` в Firebase Firestore.  
Каждый документ содержит поля: `age`, `course`, `avg_grade`, `meals_per_day`, `skip_breakfast`, `eat_schedule`, `fastfood_freq`, `fastfood_reason`, `drinks`, `concentration`, `food_effect`, `worst_time`, `improvement`, `comment`, `timestamp`.

## Безопасность

- Файл `serviceAccountKey.json` добавлен в `.gitignore` и **не публикуется в репозитории**
- Для деплоя на Streamlit Community Cloud ключ передаётся через переменную окружения `FIREBASE_KEY` в разделе Secrets

## Развёртывание (Streamlit Community Cloud)

1. Загрузить код на GitHub (без `serviceAccountKey.json`)
2. Подключить репозиторий на https://share.streamlit.io/
3. В настройках приложения добавить Secret:
   ```
   FIREBASE_KEY = <содержимое serviceAccountKey.json>
   ```
